# Desafio Final Módulo 5 - FRONTEND

# Sobre o projeto

Uma aplicação web perfeita para finalizar o curso de Desenvolvimento de Software da Cubos Academy com chave de ouro, envolvendo todos os conteúdos abordados durante os 8 meses. Nela o usuário é capaz de se cadastrar, efetuar o login e editar os dados do seu perfil como cpf, telefone... A aplicação simula um sistema interno de uma empresa onde é possível gerenciar os seus clientes e as suas cobranças, além de verificar quantos e quais são os clientes "Inadimplentes" e "Em dia" tembém é possível ver quantas e quais são as cobranças "Pagas", "Pendentes" e "Vencidas"

**URL da aplicação funcionando**: https://sgf-customers.vercel.app/

# Technologies

- React Native
- Date-fns
- Axios
- PostgreSQL
- Express
- JsonWebToken
- SecurePassword
- Cors
- Dotenv
- Knex
- Yup
